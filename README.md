# LaTeX-шаблон для выпускной квалификационной работы (ВКР)

Этот репозиторий содержит шаблон для подготовки выпускных квалификационных работ (бакалаврских, специализированных, магистерских) в российских вузах. Шаблон основан на стандартном классе `report` и адаптирован для компиляции с помощью **LuaLaTeX** или **XeLaTeX**, что обеспечивает полноценную поддержку кириллицы и современные возможности набора.

## Возможности

- Полностью настроенный класс `diplom.cls` с соблюдением российских требований к оформлению.
- Автоматическая генерация титульного листа, реферата и содержания.
- Поддержка русского языка через пакеты `babel` и `fontspec`.
- Основной шрифт — Times New Roman, моноширинный — DejaVu Sans Mono.
- Поля страницы: левое 3 см, правое 2 см, верхнее 2 см, нижнее 2 см; полуторный межстрочный интервал.
- Правильное форматирование заголовков глав, разделов и подразделов.
- Оформление подписей рисунков и таблиц по ГОСТ.
- Автоматический подсчёт страниц, глав, рисунков, таблиц, приложений и источников.
- Интеграция с `biblatex` и стилем `gost-numeric` для списка литературы.
- Готовые примеры:
  - рисунков (включая TikZ и внешние изображения),
  - таблиц,
  - математических формул,
  - алгоритмов (пакет `algorithm2e`),
  - листингов кода (пакет `listings`),
  - приложений.

## Требования

Для компиляции документа необходима **LaTeX-система**, содержащая:

- движок **LuaLaTeX** (рекомендуется) или **XeLaTeX**;
- программу **Biber** для обработки библиографии;
- следующие пакеты (обычно входят в полные дистрибутивы TeX Live или MiKTeX):
  - `xparse`, `geometry`, `setspace`, `indentfirst`, `titlesec`, `titletoc`, `caption`, `enumitem`
  - `amsmath`, `amssymb`, `mathtools`, `amsthm`
  - `totcount`, `hyperref`, `booktabs`, `graphicx`, `letltxmacro`
  - `fontspec`, `babel`
  - `biblatex` (со стилем `gost-numeric`)
  - `tikz` (с библиотеками `positioning`, `arrows.spaced`, `calc`, `shapes.geometric`)
  - `algorithm2e`, `listings`
  - `icomma`, `microtype` (опционально)

## Быстрый старт

1. **Склонируйте** или **скачайте** этот репозиторий.
2. Отредактируйте `main.tex`, указав свои данные:
   - Название работы: `\Title{Название}`
   - Автор: `\Author{Статус студента}{И.И. Иванов}`
   - Научный руководитель: `\Chief{Статус руководителя}{П.П. Петров}`
   - Заведующий кафедрой: `\ZavKaf{Статус}{С.С. Сидоров}`
   - Направление подготовки: `\Napr{...}`
   - Кафедра: `\Kafedra{...}`
   - Ключевые слова: `\Keywords{...}`
   - Город и год (опционально): `\City{Город}`, `\Year{год}`
3. Напишите текст работы, разбивая его на главы и разделы.
4. Добавьте библиографические записи в файл `references.bib`.
5. Скомпилируйте документ.

### Компиляция (LuaLaTeX + Biber)

В терминале выполните команды:

```bash
lualatex main.tex
biber main
lualatex main.tex
lualatex main.tex   # второй запуск для разрешения перекрёстных ссылок
```

Если вы используете **XeLaTeX**, замените `lualatex` на `xelatex` (но учтите, что класс оптимизирован для LuaLaTeX).

В большинстве современных редакторов (TeXstudio, TeXmaker, VS Code с расширением LaTeX Workshop) можно настроить цепочку компиляции: `lualatex -> biber -> lualatex -> lualatex`.

## Настройка

### Титульный лист

Титульный лист формируется командой `\maketitle`. Данные для него задаются в преамбуле `main.tex`:

- `\LetterHead{...}` — полное наименование вуза (по умолчанию — ЯрГУ им. П.Г. Демидова)
- `\Kafedra{...}` — название кафедры
- `\Napr{...}` — направление подготовки / профиль
- `\ZavKaf{статус}{имя}` — заведующий кафедрой
- `\DocumentType{...}` — тип документа (например, «Выпускная квалификационная работа»)
- `\Title{...}` — название работы (можно разбить на строки)
- `\Author{статус}{имя}` — статус студента и его ФИО
- `\Chief{статус}{имя}` — статус научного руководителя и его ФИО
- `\City{...}` — город (по умолчанию «Ярославль»)
- `\Year{...}` — год (по умолчанию текущий)

Если требуется оформить **курсовую работу**, добавьте `\Kursovaya` перед `\maketitle` — это немного изменит титульный лист.

### Реферат

Реферат создаётся с помощью окружения `abstract`. Ключевые слова задаются командой `\Keywords{...}`. В реферате автоматически отображаются общее количество страниц, глав, рисунков, таблиц, источников и приложений.

### Главы и разделы

Используйте стандартные команды `\chapter{...}`, `\section{...}`, `\subsection{...}`. Для ненумерованных глав (введение, заключение и т.п.) применяйте `\chapternonum{...}`.

### Приложения

Приложения размещаются после команды `\appendix`. Каждое приложение оформляется как `\chapter{...}`. Класс автоматически присваивает им обозначения «Приложение А», «Приложение Б» и т.д.

### Библиография

Библиография подключается через `biblatex` со стилем `gost-numeric`. Добавляйте источники в файл `references.bib` и цитируйте их с помощью `\cite{ключ}`. Список литературы выводится командой:

```latex
\printbibliography[title={Список литературы}]
```

Количество использованных источников подсчитывается автоматически и отображается в реферате. Источники, в тексте работы на которые отсутствует ссылка, не учитываются и не попадают в итоговый список литературы. Порядок источников определяется порядком цитирования их в тексте работы.

### Рисунки и таблицы

- **Рисунки** — окружение `figure`. Подпись располагается под рисунком и оформляется по образцу: «Рис. 1 – Название».
- **Таблицы** — окружение `table`. Подпись располагается над таблицей: слово «Таблица» и номер выравниваются вправо, а название — отдельной строкой по центру.

Пример рисунка:

```latex
\begin{figure}[!ht]
  \centering
  \includegraphics[width=0.3\textwidth]{apple.jpg}
  \caption{Яблоко}
  \label{fig:apple}
\end{figure}
```

Пример таблицы:

```latex
\begin{table}[tbh]
  \caption{Сравнение}
  \label{tab:compare}
  \centering
  \begin{tabular}{lccc}
    \toprule
    Элемент & A & B & C \\
    \midrule
    1 & 10 & 20 & 30 \\
    \bottomrule
  \end{tabular}
\end{table}
```

### Алгоритмы и листинги

Для псевдокода используется пакет `algorithm2e` (см. пример в `main.tex`, алгоритм `\ref{alg:quicksort}`).

Листинги кода набираются пакетом `listings`. В примере в `main.tex` в приложение включён файл `sort.cpp` с кодом на C++.

### Рисунки в TikZ

Пакет `tikz` позволяет создавать векторные рисунки. В `main.tex` приведены два примера: блок-схема и график функции.
